# CampusHub: Multi-tenant Student & Library Management System

CampusHub is a powerful, production-ready platform designed for modern libraries, study centers, and co-working spaces. It leverages a robust multi-tenant architecture to provide isolated management environments for multiple institutions.

## üöÄ Core Features

### For Administrators
- **Admin Dashboard**: Visualize library performance with income/expense charts and real-time KPI tracking.
- **Student Management**: Full CRM capabilities to track student status, history, and archival.
- **Visual Seating Plan**: Manage room layouts and time-bound seat assignments with tiered pricing (Basic, Standard, Premium).
- **Billing & AI Receipts**: Automated bill generation with professional, AI-formatted receipts powered by Google Genkit.
- **Expense Tracking**: Monitor operational costs with category-based breakdown charts.
- **Print Queue**: Manage and process "Print on Desk" requests from students.
- **Secure Invites**: Control library growth with single-use or time-limited invite codes.
- **Ownership Transfer**: A secure, two-step process for transferring library administrative rights.

### For Students
- **Student Dashboard**: View upcoming bookings, pending bills, and payment streaks.
- **Print on Desk**: Upload documents for printing directly from the student portal.
- **Feedback Loop**: Submit suggestions and feedback via the digital suggestion box.
- **Onboarding**: Securely join a specific library using an administrator's invite code.
